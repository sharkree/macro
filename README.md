# Macro

This is our HackCMU 2025 project that helps CMU students find campus meals that align with their nutrition goals.

## Overview

Typically, it is difficult to track calories and macronutrients in a traditional buffet-style college dining halls. CMU has restraunt-style dining, which gives us the unique opportunity for students to actually control what they are eating. Macro simplifies this process by generating meal recommendations based on a user's height, weight, and fitness goals. We utilize Next.js and MongoDB in this project.

## Features

* Input height, weight, and fitness goals
* Estimate daily nutritional requirements
* Generate meal combinations from campus dining options
* Display nutritional information for recommended meals
* Help students make more informed dining choices

## Limitations

This project was developed during a hackathon with limited data and currently supports only a subset of campus dining locations (ABP).
